# Network Stats

# Installation
```
$ git clone https://github.com/Lunik/network-stats.git
$ cd network-stats
$ npm install
```

# Run
Scrape the network
```
$ npm run scrape
or
$ npm run forever-scrape
```
Run the web server
```
$ npm run start
or
$ npm run forever-start
```
